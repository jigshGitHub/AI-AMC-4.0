3. Agent Tool-Calling Loop (Step by Step)
STEP 1 -- RECEIVE
User sends a message to the agent
v
REPEATS UNTIL LLM GIVES FINAL ANSWER
STEP 2 -- THINK
LLM reads messages + tool descriptions
and decides what to do next
v
STEP 3 -- ACT
LLM calls a tool with specific arguments
(e.g., draft_email with the user's idea)
v
STEP 4 -- OBSERVE
Tool runs and returns its result
Result is added to the message history
v
STEP 5 -- RESPOND
LLM returns a final text answer to the user

5. Key LangChain Concepts Used
ChatOpenAI
The LLM wrapper. Sends prompts to OpenAI's API and returns responses. Like a phone line to GPT.
from langchain_openai import ChatOpenAI
PromptTemplate
A reusable template with {placeholders} that get filled before sending to the LLM. Like a form with blanks.
from langchain_core.prompts import PromptTemplate
@tool Decorator
Turns a Python function into a tool the agent can call. The docstring becomes the tool's description -- the agent reads it to decide when to use it.
from langchain_core.tools import tool
create_agent
Combines an LLM + tools + system prompt into a runnable agent. The agent loops: call LLM --> call tool --> feed result back --> repeat until done.
from langchain.agents import create_agent
System Prompt
Instructions that tell the agent its role and behavior. Like giving an employee a job description before they start work.
HumanMessage
Represents the user's input in LangChain's message format. The agent receives a list of messages and responds accordingly.
from langchain_core.messages import HumanMessage
